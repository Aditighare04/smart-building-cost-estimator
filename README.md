# Smart Building Cost Estimation

## Overview
This project predicts construction cost using Machine Learning and provides a chatbot for guidance.

## Features
- ML-based prediction (Random Forest)
- Chatbot assistant
- Modern UI with Streamlit
- Real-world dataset (synthetic)

## Tech Stack
- Python
- Streamlit
- Scikit-learn

## Run Project

pip install -r requirements.txt

cd model
python train_model.py

streamlit run app.py

## Deployment
- Streamlit Cloud
- Hugging Face Spaces

## Use Cases
- Builders
- Engineers
- Real estate planners
