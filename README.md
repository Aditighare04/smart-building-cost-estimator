# 🏗 Smart Building Cost Estimation System

## 📌 Overview
This project predicts building construction cost using Machine Learning.

## ⚙️ Configuration Steps

### 1. Clone the Repository
git clone https://github.com/your-username/smart-building-cost-estimator.git
cd smart-building-cost-estimator

### 2. Install Libraries
pip install -r requirements.txt

### 3. Train Model
python -m model.train_model

### 4. Run App
python -m streamlit run app.py

### 5. Open Browser
http://localhost:8501
